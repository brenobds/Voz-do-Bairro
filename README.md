# Voz do Bairro — Protótipo Expo

App móvel simples para relatar problemas de impacto social e cidadania.

Como rodar (no seu computador com Node.js e Expo instalado):

1. Instalar dependências:

```bash
npm install
```

2. Rodar o projeto (Expo):

```bash
npm start
```

Observações:
- Este protótipo usa `AsyncStorage` para persistência local dos relatos.
- Estilização utiliza `nativewind` (Tailwind para React Native). Se quiser testar no Web, alguns pacotes nativos podem precisar de ajustes.
